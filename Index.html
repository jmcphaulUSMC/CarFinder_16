About

Car Finder is a fully functioning WebAPI web application. It is composed of a dynamic front-end using AngularJS, HTML5 and Twitter Bootstrap.

It is used to search for cars by using the year, make, model, and trim. This Application has the ability to sort and filiter results, makes finding your car incredibly simple!!!

This appplication accesses the NHSA's car recall database using the provided WebAPI. Car finder also includes images found using Bing's Search WebAPI and dynamically formats itself based on available content and device screen size.

Find You Car!

A special thanks to WebAPI.

Be mindful that some cars has no image or Recalls Infomation!!!.
