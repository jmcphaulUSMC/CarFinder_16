About

About Car Finder is a fully functioning WebAPI web application. It is composed of a dynamic front-end using AngularJS, HTML5 and Twitter Bootstrap.

It is used to search for cars by using the year, make, model, trim available. The ability to sort and filiter results makes finding your car incredibly simple!!!

This appplication accesses the NHSA's car recall database using the provided WebAPI to how car recalls. Including images found using Bing's Search WebAPI. The applicaiton dynamically formats itself based on available content and device screen size.
